## 🎯 À propos de ce projet

Ce portfolio a été développé sur mesure pour présenter mes compétences techniques (Électronique, Informatique Industrielle, Automatisme), mes projets académiques (SAÉ) et mon parcours professionnel.

### 🛠️ Technologies et Architecture
- **HTML5 & CSS3** : Structure sémantique, design moderne "Tech" et conception responsive (adaptée aux mobiles).
- **JavaScript (Vanilla)** : Développement d'une architecture de type **SPA (Single Page Application)**. La navigation entre l'accueil, le parcours et les détails des projets se fait de manière instantanée, sans rechargement de page.
- **Hébergement** : Déployé via GitHub Pages.

## 🔒 Confidentialité & Cybersécurité

Conscient des enjeux liés à l'exposition des données personnelles sur internet :
1. Le CV disponible en téléchargement est une version "Web" allégée (sans adresse postale précise ni numéro de téléphone).
2. Un fichier `robots.txt` est configuré pour limiter l'indexation directe des documents PDF par les moteurs de recherche.
